 🎓 Virtual Classroom Management System

A web-based Virtual Classroom Management System developed using Python, Flask, HTML, CSS, JavaScript, SQLite, and Machine Learning. The system provides an interactive and intelligent learning environment by automating classroom management, attendance tracking, student engagement monitoring, and academic performance prediction.

 📌 Project Overview

The Virtual Classroom Management System is designed to improve the teaching and learning experience through a user-friendly web application. It offers separate dashboards for students, faculty, and administrators to efficiently manage classroom activities.

The system integrates Machine Learning (Random Forest Classifier) to predict student performance based on attendance, academic marks, and engagement levels. This helps faculty identify students who may require additional academic support and enables early intervention.

🧠 Machine Learning Module

The system uses the Random Forest Classifier algorithm to predict whether a student is Safe or At Risk based on the following parameters:

- Attendance Percentage
- Academic Marks
- Student Engagement Score

The prediction results help faculty monitor student progress and provide timely academic guidance.
